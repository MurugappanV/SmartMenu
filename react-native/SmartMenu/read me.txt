---Smart Menu - React Native App---

---Execute commands in CLI - Project root folder

Initiation 
    new project
        react-native init SmartMenu            react-native init <Project Name>
    downloaded project 
        npm install 

run android
    react-native run-android


NPM packages used 
    Bottom nagivation 
        yarn add react-native-material-bottom-navigation
        https://www.npmjs.com/package/react-native-material-bottom-navigation#install

    react navigation
        yarn add react-navigation
        https://reactnavigation.org/docs

    vector icons
        yarn add react-native-vector-icons
        https://github.com/oblador/react-native-vector-icons
        inner requirements
            add following line in android\app\build.gradle
            apply from: "../../node_modules/react-native-vector-icons/fonts.gradle"

    Colors Material Design
        https://material.io/guidelines/style/color.html#color-color-palette
